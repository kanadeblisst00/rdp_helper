## 功能
![](http://cdn.ikanade.cn/202311241200326.png)

通过你给定的连接信息帮你创建rdp文件。

## 解决的需求场景

为什么需要它？电脑自带的记住密码一个IP只能记住一个用户名，如果端口变了，它还是用的这个用户名和密码。

另外，我需要一个快捷方式打开它，而不是每次都要输入IP。

不过，这里推荐微软官方的[`Remote Desktop Connection Manager`](https://learn.microsoft.com/en-us/sysinternals/downloads/rdcman)更好用
